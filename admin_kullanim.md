# V26 Admin Panel Kullanımı

Bu panel şu anda tarayıcı içinde yerel taslak mantığıyla çalışır.

1. Yönetim bölümüne gir.
2. Marka bilgilerini doldur.
3. Yerel taslak kaydet.
4. Tüm veriyi JSON indir.
5. JSON dosyasını proje `data.json` dosyası olarak kullanabilir veya Supabase'e aktarabilirsin.

Sonraki aşama V27:
- Supabase'e doğrudan kaydetme
- Admin giriş kontrolü
- Görsel yükleme
