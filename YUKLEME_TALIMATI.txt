Yükleme:
1. ZIP içindeki tüm dosyaları GitHub projenin ana klasörüne kopyala.
2. Mevcut dosyaların üzerine yaz.
3. Terminalde:

git add .
git commit -m "V44 hukuki bilgilendirme ve onay ekrani"
git push

4. GitHub Pages güncellenince Ctrl+F5 ile yenile.

Not:
Kullanıcı ilk açılışta bilgilendirme ekranını kabul etmeden uygulamayı kullanamaz.
