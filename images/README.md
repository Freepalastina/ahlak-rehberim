# Marka Görselleri

Buraya kendi eklediğin veya kullanım izni olan marka/ürün görsellerini koyabilirsin.

Örnek:
- images/nutella.png
- images/coca-cola.png

Sonra `data.json` içindeki ilgili kayda şunu yaz:
```json
"image_url": "images/nutella.png"
```

Otomatik internetten logo çekmek yerine, izinli veya kendi yüklediğin görselleri kullanmak daha güvenlidir.
