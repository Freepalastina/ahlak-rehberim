# V27 Supabase Admin Sync

1. Supabase SQL Editor'de `supabase_v27_schema.sql` dosyasını çalıştır.
2. Uygulamada Yönetim paneline gir.
3. Supabase URL ve publishable/anon key gir.
4. Tablo adı olarak `brands` kullan.
5. Bağlantıyı test et.
6. Bir marka formu doldurup “Bu Kaydı Supabase'e Gönder” butonuna bas.

Önemli:
- service_role key kullanma.
- Public anon key veya publishable key kullan.
- Yazma izni için RLS policy gerekir.
